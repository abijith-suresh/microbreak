# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.0.5] - 2026-04-24

### Added

- Repo hygiene baseline with CI, PR title checks, Dependabot, Husky hooks, linting, formatting, and a changelog.

### Changed

- Standardized the project on a Bun-first quality gate and conventional commit workflow.

## [0.0.4] - 2026-04-24

### Added

- Public open-source project metadata, MIT licensing, and Bun-based Vercel deployment configuration.
- Stronger repo validation with Astro check and reference-backed Sudoku regression tests.

### Fixed

- Sudoku loading state transitions so a started game reliably reaches a fully rendered board.

## [0.0.3] - 2026-04-24

### Added

- A polished Sudoku setup flow, completion state, and refined client-side game experience.

### Changed

- Locked Sudoku board size and difficulty choices to the pre-game setup phase.

## [0.0.2] - 2026-04-24

### Added

- The first complete Microbreak implementation with a playable Sudoku experience and supporting design system.

## [0.0.1] - 2026-04-24

### Added

- Initial Astro project scaffold.
